Weekend project turned World Cup preview:

I trained a match prediction pipeline in Python for Mexico vs Ecuador (Round of 32).

Stack:
• pandas + scikit-learn
• Feature engineering: Elo, 10-game rolling form, H2H, tournament weighting
• GradientBoostingClassifier (3-class outcome)
• HistGradientBoostingRegressor (home/away goals)
• Poisson matrix for scoreline probabilities

Results on holdout set:
• 57.2% outcome accuracy
• 1.07 MAE home goals / 0.84 MAE away goals

Today's forecast:
Mexico 66.7% | Draw 17.4% | Ecuador 15.9%
Top scoreline: 1-0 (17.5%)

Code + data pipeline are reproducible end-to-end.
